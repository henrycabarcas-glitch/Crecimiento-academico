# **App Name**: Scholarly Growth

## Core Features:

- Student Records Management: Securely manage student data, including enrollment, demographics, and academic history, complying with Colombia's Law 1581.
- Curriculum Planning: Enable teachers to create and manage lesson plans according to the MEN standards, linking learning objectives to evaluations.
- Digital Gradebook: Teachers can record qualitative and quantitative assessments, achievements, and competencies.
- Automated Report Card Generation: Automatically generate report cards compliant with Colombian regulations.
- Behavioral Observation Log: Track and manage student behavior, disciplinary actions, and early alerts for school counseling, according to school's code of conduct.
- Fee Management & Billing: Handle tuition payments, billing, and generate financial reports. Includes role-based access for administrators and payment tracking for parents.
- Parent Communication Portal: Parents can access grades, report cards, and communicate directly with teachers. The LLM tool provides suggested messages to encourage interaction and engagement with school activities.

## Style Guidelines:

- Primary color: Warm and inviting blue (#64B5F6) to evoke trust and reliability.
- Background color: Soft light-blue background (#E3F2FD) to create a calm and approachable environment.
- Accent color: Muted green (#81C784) for positive actions and highlights.
- Body and headline font: 'PT Sans', a humanist sans-serif.
- Use clean and simple icons for navigation and features, ensuring ease of understanding for all user roles.
- Design a responsive layout with clear information hierarchy for easy navigation on all devices.
- Subtle animations to provide feedback on interactions and guide users through processes.
